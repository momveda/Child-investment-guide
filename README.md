# Claude Master Prompt for Work, Home & Motherhood

This repository shares a **calm, paste‑ready master prompt** designed to help manage **work, home, and personal well‑being** using Claude — without overwhelm, hustle culture, or unrealistic productivity expectations.

It is a **human‑first framework**, created for real life, not ideal days.

---

## What this is

This project includes:

- A simple public webpage that shares a base Claude prompt  
- A prompt you can copy, paste, and adapt for your own life  
- A structure that works even if you are not technically inclined  

You do **not** need any technical background to use this.  
If you can use Claude, you can use this prompt.

---

## Who this is for

This prompt is especially helpful if you are balancing:

- Professional work alongside home and family responsibilities  
- Mental and emotional load across a full, layered day  
- The desire for clarity rather than pressure  
- Using AI as support, not as another obligation  

It was intentionally designed with **mothers and caregivers** in mind, though it can be adapted for anyone living full, interconnected lives.

---

## How to use it

1. Open Claude  
2. Copy the prompt from the main page  
3. Paste it into Claude and customize it to fit your reality  

There is **no setup**, **no tooling**, and **no learning curve**.

---

## A gentle note on customization

This is a **base version** of the prompt. It is intentionally simple and flexible.

You are encouraged to:

- Change wording  
- Remove what does not apply  
- Add personal context  
- Let it evolve as your season of life changes
